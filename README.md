# BoxCart-Gantry

The BoxCart-Gantry is a cartesian gantry for voron trident-like printers. Specifically for large print sizes(>350mm).
### Files
* [CAD](/BoxcartGantry.step)
* [BOM(csv)](Bom.csv)

### Project goals:
- Short belt paths
- Rigid components
- Low component count
- High performance
